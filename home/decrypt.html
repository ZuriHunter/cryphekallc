Encryption

Step 1

Enter the unique key that was given to you by the sender in the "Special Key" box. Paste in the encrypted message within the message.

Step 2

If you receive an encrypted file. Upload the file and select in the box towards the right, what type of file it is.

Step 3

After all fields are required select the "Decrypt" button so you can read your message.
